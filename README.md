# WEB-приложение по продаже автомобилей
При создании приложения использованы технологии:
 * HTML5, CSS3 + flexbox, JQuery
 * VCS Git
 * Java 14
 * Apache Maven
 * Apache Tomcat 9
 * Servlet Api, JSTL
 * PostgreSQL
 * JPA, Hibernate
 * Travis-ci
